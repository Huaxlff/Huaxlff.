输出二叉树的前中后序遍历

题目大意

给出一棵二叉树，分别输出先序、中序、后序遍历结果。

输入

第1行 : 结点数 n(1<=n<=1e6)n (1 <= n <= 1e6)n(1<=n<=1e6)
以下 nnn 行,每行 333 个整数，分别表示父结点、左孩子、右孩子。若没有孩子，对应的整数为 000.

输出

第111行 : 树根
第222行 : 先序遍历结果，数字间用 111 个空格分开。
第333行 : 中序遍历结果，数字间用 111 个空格分开。
第444行 : 后序遍历结果，数字间用 111 个空格分开。

样例输出
